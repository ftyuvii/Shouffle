Privacy Policy

Data Management

Shouffle processes core Discord metadata (IDs for users, roles, and guilds) to preserve custom module rules such as logging configurations, custom responses, and welcome channels. No external tracking logs or data records are created.

Temporary Context

Dynamic event actions (such as tracking a recently deleted message block for the short-term use of snipe) run purely inside standard temporary caches and are discarded automatically.

User Agency

Server managers hold absolute command. Deactivating active frameworks or utilizing commands like botlogs disable deletes related configurations immediately.
