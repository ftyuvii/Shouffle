🌸 Discord Developer Compliance: This policy is designed to fully comply with Discord's Developer Terms of Service and Developer Policy rules regarding clear user data disclosures.

Data We Collect & Process

Shouffle operates using minimal data tracking required to execute standard Discord bot features:

  • User Identifiers: Your unique Discord User ID, Username, and Server/Guild IDs are read to process commands, enforce access permissions, and customize server features.
  • Command Context Data: When you execute a command, the text content of that specific command, channel ID, and timestamp are processed in real-time to generate a prompt response.

Data Storage & Retention

We do not store your private conversations or structural server layouts. Any configuration data (such as custom bot prefixes, server settings, or toggles) is stored securely in our database. This structural configuration data is retained until the Bot is removed from your server or until a request for manual erasure is made.

Data Deletion & Rights

You retain full ownership and control over your data. If you wish to have your server configurations or user-associated data permanently deleted from our system, you can remove the Bot from your server or contact the support team directly. All configuration data related to that server will be fully wiped out within 30 days of uninstallation.
