Quick Start Guide

For beginners ✿
<argument>

Required — you must fill this in to run the command.

[argument]

Optional — you can leave this blank if you want.

?command

Prefix style — type directly in chat with the ? symbol.

/command

Slash style — Discord's built-in autocomplete menu.

?help

Shows all available commands inside the bot's help panel.

?botinfo

See Shouffle's stats, uptime, and build information.

☁️ general & utility 27 commands

ping
Tests the connection speed between Discord and Shouffle.
uptime
Shows how long Shouffle has been online without interruption.
botinfo
Displays details about Shouffle, including statistics and build info.
serverinfo
Provides a neat breakdown of the server's owner, age, and features.
roleinfo
Shows deep configurations, permissions, and settings for a specific role.
membercount
Counts the total number of real users and bots in the server.
inviteinfo
Analyzes a server invite link to find its origin and creator details.
emojiinfo
Reveals ID codes and direct image assets for any server emoji.
stick
Pins a chosen text to the bottom of a text channel dynamically.
stickremove
Removes the current active sticky message from the channel.
autoresponder
Sets up Shouffle to automatically reply to custom keyword triggers.
removeresponder
Deletes an active autoresponder keyword from the trigger list.
listresponders
Displays all custom auto-replies currently active in the server.
say
Forces the bot to mirror and send exactly what text you type.
embed
Builds stylized, structured text blocks using rich card layouts.
roll
Generates a randomized dice roll outcome for mini-games.
coinflip
Flips a virtual coin to settle simple server choices (Heads/Tails).
choose
Randomly picks an option from multiple choices you provide.
8ball
Answers questions with random fortune-telling logic outcomes.
calculate
Acts as a direct math solver inside the chat interface.
snipe
Displays the content of the last deleted message in the channel.
steal
Clones external custom emojis or stickers to your server.
avatar
Grabs the high-resolution profile icon of any designated user.
banner
Grabs the high-resolution background banner of a user.
afk
Sets a custom status notice when you are away from keyboard.
afkremove
Clears your active AFK status immediately upon returning.
poll
Creates a custom public reaction poll for members to cast votes.

🛡️ moderation operations 16 commands

mute
Applies a text and voice channel timeout to a target member.
unmute
Removes any active mute timeout restrictions from a profile.
kick
Removes a target user from the server while leaving an open rejoin route.
softban
Bans and instantly unbans a user to wipe their recent message history.
ban
Permanently restricts a user ID from entering the server.
unban
Pulls a user ID out of the server's central ban ledger.
warn
Logs a recorded infraction point on a user profile for rule tracking.
clearwarns
Resets and deletes all logged warning records for a user.
purge
Deletes a specified number of messages from the channel quickly.
mediaonly
Locks channels to automatically drop text without links or files.
unmediaonly
Reverts channels to allow a normal mix of text and files.
roleadd
Assigns an existing server role directly to a member's profile.
roleremove
Strips an active server role off a target member's profile.
deafen
Mutes all audio reception for a user inside active voice chats.
undeafen
Restores regular voice listening for a deafened user.
vcban
Disconnects and blocks a user from joining any voice channels.

autorole

vcsetup
